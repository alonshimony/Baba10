Ten Years of Baba — Setup

← view the site

Drag photos onto a year (multiple at once works) or onto a single slot to replace it, write the speech-bubble text under each photo, then hit Save to site folder and pick the baba-ten folder — the photos are auto-cropped to 4:3, shrunk if too big, written into photos/, and content.json is generated for you.

☁️ Cloud setup — auto-update the live site (Vercel + GitHub)

When the site is hosted on Vercel from a GitHub repo, saving here commits the photos and content.json straight to GitHub — Vercel then redeploys automatically and the live site updates in ~a minute. Create a fine-grained token limited to that one repo with Contents: Read and write permission.
